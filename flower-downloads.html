🌼 Flower Images Download

Download free stock images for the Alberta Bee Adventure flora section

📥 How to Download:

  1. Click on any link below (Wikimedia, Unsplash, or Pexels)
  2. Right-click the image → "Save Image As..."
  3. Save it with the exact filename shown (e.g., canola.jpg)
  4. Place it in: public/images/flora/
  5. Repeat for all 8 flowers
  6. Commit and push to GitHub
